# Портфолио студента-разработчика

Личный веб-сайт студента-разработчика, представляющий собой многостраничное портфолио, которое сочетает в себе функции визитной карточки, галереи проектов и учебного дневника и много других приколов.

## Структура проекта

```
base-project/
├── index.html              # Главная страница (визитная карточка)
├── pages/                  # Директория дополнительных страниц
│   ├── projects.html       # Страница проектов с модальными окнами
│   ├── diary.html          # Учебный дневник с прогресс-барами
│   └── contacts.html       # Страница контактов с валидацией формы
├── styles/                 # Директория стилей
│   └── styles.css          # Основной файл стилей
├── images/                 # Директория изображений
│   ├── icons/              # Поддиректория для иконок
│   ├── photo.png           # Фото студента
│   └── project*.jpg        # Изображения проектов
├── scripts/                # Директория скриптов (для КР3+)
├── assets/                 # Дополнительные ресурсы
│   └── resume.pdf          # Резюме для скачивания
└── README.md               # Документация проекта
```

## Описание страниц

### Главная страница (index.html)
Визитная карточка студента с информацией о себе, навыках и быстрыми ссылками на другие разделы сайта.

Основные элементы:
- Фото профиля
- Информация о студенте
- Бейджи навыков
- Прогресс-бары навыков
- Карточки быстрого доступа к разделам

### Страница проектов (pages/projects.html)
Коллекция учебных и личных работ студента в виде адаптивной сетки.

Основные элементы:
- Сетка карточек проектов
- Модальные окна с детальной информацией о каждом проекте
- Описание, технологии, ссылки на живую версию и исходный код

### Учебный дневник (pages/diary.html)
Отслеживание образовательного прогресса студента.

Основные элементы:
- Текущие курсы с прогресс-барами
- Завершенные курсы
- Статистика обучения

### Страница контактов (pages/contacts.html)
Контактная информация и форма обратной связи.

Основные элементы:
- Контактная информация
- Форма обратной связи с валидацией
- Социальные сети

## Технологии
